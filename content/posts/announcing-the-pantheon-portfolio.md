I'm building The Pantheon with a modern, powerful, and delightfully type-safe stack:

- **Framework:** Next.js 15 (App Router), React 18, and TypeScript. Because I like my code fast, flexible, and free of unexpected `undefined` errors.
- **Styling:** Tailwind CSS v4. Because utility-first CSS is the only way to fly.
- **3D (Coming Soon):** React Three Fiber, Three.js, and GSAP. Because a temple needs a grand, animated entrance.
- **AI (Planned):** The Groq API for lightning-fast inference, a RAG pipeline with a vector database, and four custom-built, personality-driven chatbots (one for each god, naturally).
